Setup and Game Assets

By now you should have downloaded and installed to mu_code the ZIP archive of game assets. Using the ZIP archive should simply things, but if you want, you can install the assets for our Fruit Ninja games individually using the following links.

  • Change directory to your standard folder for python projects. It is probably mu_code.
  • Make a new sub-directory called fruit_ninja_games.
  • Inside sub-directory fruit_ninja_games, make a new sub-directory called images.
  • Download (right click on) each of the following images and place them into the images directory.
    • apple.png 
    • banana.png 
    • burger.png 
    • cherries.png 
    • fries.png 
    • grapes.png 
    • lemon.png 
    • pear.png 
    • pineapple.png 
    • pizza.png 
    • soda.png 
    • tomato.png 
  • Inside sub-directory fruit_ninja_games, make a new sub-directory called sounds.
  • Download (right click on) each of the following images and place them into the sounds directory.
    • shoot.wav
    • shoot_and_cheer.wav
  • Inside sub-directory fruit_ninja_games, make a new sub-directory called music.
  • Download (right click on) each of the following images and place them into the music directory.
    • the_french_lovers_song.mp3
    • trap_ashamaluev_music.mp3
